After the survey, I found some interesting user information:

  • Most users are between the ages of 18 and 30.
  • Most users are students, and their monthly salary ranges from 7001 to 15,000.
  • Most membership card users have both physical and electronic membership cards.
  • The main problem while using membership cards is forgetting the card, inconvenient to carry multiples card and spend a long time searching for the card.
  • The main problem while using membership applications is spending a long time accessing the application.
  • There are too many channels available to find out about the store's offer.
ABOUT THE PROJECT
Pocket Cards is a mobile wallet application that allows users to store loyalty cards and view membership store promotions in a single app on a smartphone.
The goal of this team project is to learn about the design thinking process, cooperation, workshops, and brainstorming as part of the Designing Interactive Systems course.
