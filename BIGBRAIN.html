Overview
The Logo
This project is about applying the knowledge gained from studying subjects such as Gestalt theory, Design Principles and Phycology of color to web design.

Gestalt theory says our brain likes to organize visual information into "figure" and "background". We can use this in design to make things interesting and help people focus on specific parts of the design.

This logo looks like a brain from the top view, representing knowledge.

The Color
Blue is often associated with feelings of calmness, professionalism, and trustworthiness, which can be appropriate for an educational website.

Affordance is the principle that means an object should clarify what it is for and how it should be used without requiring additional instructions or explanation.

  • Users know the function of the icon shopping cart with the plus symbol button.

 

Consistency is the principle that ensures that users can easily understand and predict how a system works.

  • Users know what will happen if the back button is pressed.

 

Feedback is the principle that ensures users receive feedback from their actions. As you can see, when the user puts a cursor on the button, it will grow to tell the user that it can be interacted with.

  • The course button will change color when the user puts the cursor on it.
